<p style="padding-top:0px; padding-bottom:0px; margin-top:0px; margin-bottom:15pt; line-height:1.5; letter-spacing:1px; font-family:Verdana,sans-serif,宋體; font-size:14px">
<span style="font-family:'Courier New'; padding:0px; margin:0px">system() executes a command specified in command by calling /bin/sh -c command, and returns after the command has been completed. During execution of the command, SIGCHLD will be blocked, and
 SIGINT and SIGQUIT will be ignored.</span></p>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
system()函式呼叫/bin/sh來執行引數指定的命令，/bin/sh 一般是一個軟連線，指向某個具體的shell，比如bash，-c選項是告訴shell從字串command中讀取命令；</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
在該command執行期間，SIGCHLD是被阻塞的，好比在說：hi，核心，這會不要給我送SIGCHLD訊號，等我忙完再說；</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
在該command執行期間，SIGINT和SIGQUIT是被忽略的，意思是程序收到這兩個訊號後沒有任何動作。</div>

<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
<span style="padding:0px; margin:0px">再來看一下system()函式返回值：</span></div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
<div style="padding:0px; margin:0px"><span style="font-family:'Courier New'; padding:0px; margin:0px">The value returned is -1 on error (e.g. fork(2) failed), and the return status of the command otherwise. This latter return status is in the format specified
 in wait(2). Thus, the exit code of the command will be WEXITSTATUS(status). In case /bin/sh could not be executed, the exit status will be that of a command that does exit(127).</span></div>
<div style="padding:0px; margin:0px"><span style="font-family:'Courier New'; padding:0px; margin:0px">If the value of command is NULL, system() returns nonzero if the shell is available, and zero if not.</span></div>
</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
<div style="padding:0px; margin:0px">為了更好的理解system()函式返回值，需要了解其執行過程，實際上system()函式執行了三步操作：</div>
</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
1.fork一個子程序；</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
2.在子程序中呼叫exec函式去執行command；</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
3.在父程序中呼叫wait去等待子程序結束。</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
對於fork失敗，system()函式返回-1。</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
如果exec執行成功，也即command順利執行完畢，則返回command通過exit或return返回的值。</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
（注意，command順利執行不代表執行成功，比如command："rm debuglog.txt"，不管檔案存不存在該command都順利執行了）</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
如果exec執行失敗，也即command沒有順利執行，比如被訊號中斷，或者command命令根本不存在，system()函式返回127.</div>
<div style="padding:0px; margin:0px; font-family:Verdana,sans-serif,宋體; font-size:14px">
如果command為NULL，則system()函式返回非0值，一般為1.</div>
